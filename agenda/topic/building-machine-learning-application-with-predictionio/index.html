Talk: Building Machine Learning Application with PredictionIO

Speaker: Paul Li (Hong Kong)

Community: PredictionIO

Language: English (With English Slides)

Tag: machinelearning

Beginners

About Speaker

After graduating from Carnegie Mellon University with a master degree in Electrical and Computer Engineering, Paul has worked in several big corporations in Silicon Valley and Hong Kong. Tired of the slow moving culture in big companies, he decided to work with smaller startups. Most recently, he is working with the PredictionIO team based in the Bay Area to build the best machine learning server.

About the Topic

PredictionIO is an open source machine learning server for developers and data scientists. In this presentation, we will show you how PredictionIO can help you build and deploy a machine learning applications quickly. As a demo, we will build a Sentiment Analysis application using PredictionIO and Stanford's CoreNLP library.
